
# 주석 사용법

주석은 코드의 일부로 간주되지 않으며, 코드에 대한 설명이나 메모를 추가할 때 사용합니다. 파이썬에서는 `#` 문자를 사용하여 주석을 작성합니다.

## 한 줄 주석

한 줄 주석은 `#` 뒤에 작성합니다.

```python
# 이것은 한 줄 주석입니다.
print("Hello, World!")  # 이 부분도 주석입니다.
```

## 여러 줄 주석

여러 줄 주석은 일반적으로 여러 개의 #을 사용하거나, 문자열 리터럴을 사용합니다.

```python
# 이것은
# 여러 줄 주석입니다.

"""
이것은 문자열 리터럴을 사용한
여러 줄 주석입니다.
"""
```
