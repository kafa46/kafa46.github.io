6.2. 기본 GUI 애플리케이션#

6.2.1. 윈도우 생성 및 텍스트 추가#

Tkinter 라이브러리를 사용하여 기본 윈도우를 만들고, “Hello, World!” 텍스트가 표시된 레이블을 추가한 후, 메인 이벤트 루프를 실행하여 윈도우를 표시해 봅니다.

import tkinter as tk

# 메인 윈도우 생성
root = tk.Tk()
root.title("Hello Tkinter")

# 레이블 추가
label = tk.Label(root, text="Hello, World!")
label.pack()

# 메인 루프 실행
root.mainloop()
